# Personal Portfolio / Landing Page — Uttam Pulami Magar

A **fully responsive, single-page portfolio website template** built with **pure HTML and CSS** (no frameworks).  
> **Note:** This is a portfolio **website template**. The showcased projects, skills, and tools are for demonstration purposes and do not reflect my personal skills or work experience.

It demonstrates the structure, design, and layout of a modern portfolio site, highlighting profile sections, projects, and tech stacks.
